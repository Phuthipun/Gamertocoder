รีวิวค่ายแต่ละวัน

  • Review Day 1

    วันแรกที่เรียนตื่นเต้นมากได้รู้จักพี่ๆที่เก่งมากๆหลายคนเลย เป็นการเปิดประสบการณ์ใหม่ๆได้ดีมากๆ เนื้อหา Gamification และ Introduction web Development ของวันนี้ที่พี่อ๋องและพี่เอิร์ธสอนยังไม่ตึงมากเพราะเหมือนเป็นการอินโทรต้อนรับทุกคนก่อน

  • Review Day 2

    วันที่ 2 เป็นเนื้อหาเกี่ยวกับ Inspiration ที่เป็น Section ของพี่เจที่จะคอยแนะนำให้ว่าเราควรไปสายงานไหนที่เรารู้สึกชอบ และ Section ของพีชาญที่เนื้อหาด้าน UI UX ที่เริ่มแน่นขึ้น ทำให้รู้ว่าการออกแบบก็มีหลักการของมัน

  • Review Day 3

    วันที่ 3 พี่เจมส์มาแชร์ประสบการณ์ตรงของตัวเองในการเริ่มสายงานการโค้ดของตัวเอง ได้รับความรู้ใหม่ๆเพิ่มขึ้นเยอะมากๆ และ Section ของพี่วินซี่ มาให้ความรู้ในวงการ E-sports ที่เป็นอุตสาหกรรมที่กำลังเติบโตอย่างก้าวกระโดด

  • Review Day 4

    วันที่ 4 เป็นการเริ่มเข้าเนื้อหาที่จริงจังอย่างภาษา HTML ซึ่งเป็น Section ของพี่เอิร์ธ ที่สอนการใช้ภาษา HTML ในการขึ้นโครงหน้าเว็บไซต์ หลังจากเรียนจบพวกเราสนุกมากๆเลยลองเขียนเว็บแนะนำตัวของพวกเราเอง

  • Review Day 5

    วันที่ 5 เป็นการเรียนเนื้อหาภาษา CSS ที่ใช้สำหรับตกแต่งหน้าเว็บไซต์ ขนาดเนื้อหาก็จะค่อนข้างเยอะกว่า HTML เป็น Section ของพี่เอิร์ธเช่นเคย ซึ่งหลังจบเรียนเราก็ได้นำ สิ่งที่เรียนมาทำเป็นเว็บแนะนำตัวของพวกเราได้อย่างสวยงาม

  • Review Day 6

    วันที่ 6 เป็นการเรียนเนื้อหาภาษา JavaScript เป็น Section ที่ใช้คำว่าเยอะที่สุดเลยก็ได้ เพราะมีเนื้อหายิบย่อยลงไปอีกในการเรียนวันต่อๆไป หลังจากจบเรียนเราก็ได้ศึกษาเพิ่มเติมเพื่อที่จะทำเกมง่ายๆมาเกมนึงเพื่อแกล้งเพื่อนเรา555

  • Review Day 7

    วันที่ 7 ยังคงอยู๋ในภาษา JavaScript ในส่วนของ FlowและFunction ซึ่งเนื้อหาก็เพิ่มขึ้นตามลำดับ ในตอนนี้เนื้อหาจะแน่นเป็นพิเศษและค่อนข้างงง ทำให้ตามพี่เอิร์ธไม่ทันบ้างเล็กน้อย แต่ก็พอกลับมาตามทันได้จากการดูคลิปย้อนหลัง

  • Review Day 8

    วันที่ 8 ยังคงอยู๋ในส่วนของภาษา JavaScript เป็นเนื้อหาด้าน API และเป็นการเน้นการประยุกต์ใช้ร่วมกับภาษา HTML และ CSS เป็นอีก 1 วันที่เนื้อหาแน่นสุดๆ ยังคงต้องตามเรียนจากคลิปเพราะเนื้อหาแน่นสุดๆแน่นปั้ก มีง่วงบ้างบางที แต่ต้องแอคทีฟตัวเองตลอดเวลา

  • Review Day 9

    วันที่ 9 ต่อด้วย Section ภาษา Go Language เนื้อหาวันนี้ก็จะเริ่มผ่อนๆลงมาแล้วหลังจากอัดแน่นมา 2 วันในส่วนของภาษา Go Language เป็นภาษาที่ได้ยินไม่บ่อยจากประสบการณ์ของทีม 3 คน แต่ก็ตามทันได้เพราะในทีมช่วยเหลือกัน

  • Review Day 10

    วันที่ 10 เป็นการแจ้งกฎการทำโปรเจกต์ของพี่เอิร์ธ และต่อด้วยบทเรียนสุดท้ายอีกนิดหน่อย ความรู้สึกวันสุดท้ายคือแอบใจหายที่ 10 วันที่เรียนรู้อะไรกับพี่ๆหลายคนผ่านไปเร็วมากๆ แต่ก็สนุกมากๆเหมือนกัน ลุยกับโปรเจคเรากั๊น!!

สิ่งที่ได้จากค่าย

ได้ฝึกการทำงานเป็นกลุ่มและมีทีมเวิร์คร่วมกันกับเพื่อนในทีม
สิ่งที่ได้จากค่ายนี้คือประสบการณ์ใหม่ๆและความรู้ด้านการโค้ดดิ้งที่จะนำไปต่อยอดและพัฒนาได้ พี่ๆในค่ายก็เป็นกันเองและใจดีมากๆ
ได้ฟังความรู้ในสายงานการโค้ดเป็นไกด์ในการทำอาชีพสายDeveloperในอนาคต

ช่องทางการติดต่อ

  • พุทธิพันธ์ ปานสง่า
    ตำแหน่ง : Web Developer
    รายละเอียด : นายพุทธิพันธ์ เป็นชายหนุ่มวัย 16 ปี ความสามารถพิเศษของเค้าคือการเล่น ดนตรีได้หลายชนิด ไม่ว่าจะเป็นกีตาร์ คีย์บอร์ด ขลุ่ย และอื่นๆอีกมากมาย ชายคนนี้เป็นคนที่หน้าดุ นิดหน่อย แต่ถ้ารู้จักแล้วจะใจดีมากๆ อีกทั้งยังสายเปย์อีกด้วย
  • ธนดล จันเสน
    ตำแหน่ง : Web Developer
    รายละเอียด : นายธนดล เป็นหนุ่มฮอตของสายชั้น ม.5 ด้วยหน้าตาอันแสน อ้ปป้าของเค้า ทำให้มีสาวๆเยอะมาก ความสามารถพิเศษของธนดลก็คือ การเล่นเกมที่โปรสุดๆ และยังสามารถเล่นดนตรีได้อีกด้วย
  • นภัสพร ศิริชัยนฤมิตร
    ตำแหน่ง : UX/UI Designer
    รายละเอียด : หญิงสาวคนนี้มีชื่อว่า เพลน เป็นสาวที่มากความสามารถ ไม่ว่าจะ เป็นการตีแบตที่แสนจะโหด สกิลการร้องเพลงที่เทพสุดๆ อีกทั้งยังเล่นกีตาร์ได้อีกด้วย สาวคนนี้นั้น ก็ฮอตเหมือนกัน มีคนทักมาจีบแทบทุกวัน น่าหมั่นไส้จริงๆ
